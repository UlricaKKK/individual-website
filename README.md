# Deng Kexin - Personal Portfolio Website

This is a personal portfolio website for Deng Kexin (Data Analyst), featuring a Neobrutalism design style.

## Features

- **Responsive Design**: Works on mobile and desktop.
- **Bilingual Support**: Switch between English (EN) and Chinese (中文).
- **Modern UI**: Uses Tailwind CSS with a Neobrutalism aesthetic.
- **Single Page Application**: Smooth scrolling navigation.

## How to Deploy Online (Free)

Since this project is a static website (HTML + CSS + JS), you can deploy it for free using several services.

### Option 1: Netlify Drop (Easiest)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop).
2. Drag and drop the folder containing this project (`selfmade website`) onto the page.
3. Your site will be online instantly! You can claim it to customize the URL.

### Option 2: GitHub Pages
1. Create a new repository on GitHub.
2. Push this code to the repository.
3. Go to **Settings** > **Pages**.
4. Select the `main` branch as the source and save.
5. Your site will be available at `https://<your-username>.github.io/<repo-name>`.

### Option 3: Vercel
1. Go to [vercel.com](https://vercel.com) and sign up.
2. Import your GitHub repository.
3. Vercel will automatically detect the HTML file and deploy it.

## Local Development

Just open `index.html` in your browser to view the site. No build process required as it uses Tailwind CSS via CDN.
